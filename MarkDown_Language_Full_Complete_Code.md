# 📖 Markdown Language Complete Course

Markdown is a lightweight markup language used to format text. It is commonly used for GitHub README files, documentation, blogs, and notes.

---

# 1. Headings

Use `#` symbols to create headings.

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

Output:

# Heading 1

## Heading 2

### Heading 3

---

# 2. Blockquotes

Use `>` to create blockquotes.

```markdown
> This is a blockquote.
>
> This is the second line of the blockquote.
```

Output:

> This is a blockquote.
>
> This is the second line of the blockquote.

---

# 3. Line Breaks

Add two spaces at the end of a line or use `<br>`.

```markdown
This is the first line.  
This is the second line.
```

---

# 4. Combining Headings and Blockquotes

```markdown
> ## Markdown Heading
```

Output:

> ## Markdown Heading

---

# 5. Text Formatting

## Bold Text

```markdown
**Bold Text**
__Bold Text__
```

**Bold Text**

## Italic Text

```markdown
*Italic Text*
_Italic Text_
```

*Italic Text*

## Bold and Italic

```markdown
***Bold and Italic***
```

***Bold and Italic***

---

# 6. Lists

## Unordered List

```markdown
- Item 1
- Item 2
  - Sub Item
- Item 3
```

## Ordered List

```markdown
1. First Item
2. Second Item
3. Third Item
```

---

# 7. Horizontal Lines

Create horizontal lines using:

```markdown
---
```

or

```markdown
***
```

or

```markdown
___
```

# 8. Links and Hyperlinks

## Email Address

```markdown
<example@gmail.com>
```

## Website Link

```markdown
<https://example.com>
```

## Clickable Link

```markdown
[Visit My Portfolio](https://example.com)
```

Link a PDF file:


# 10. Code Blocks

## Inline Code

```markdown
Use `print("Hello World")` to print text.
```

Output:

Use `print("Hello World")` to print text.

## Python Code Block

```python
x = 5
y = 7

total = x + y

print("Sum =", total)
```

Output:

```text
Sum = 12
```
